Backup Database

Because this CMS runs entirely in your browser without a backend server, it cannot directly generate a raw `.sqlite` file download via the Turso HTTP API.

To download a full backup, run this command in your computer's terminal:

turso db dump duasandaamalapp-alihusains > backup.sql

You can then recreate a local SQLite file using:
sqlite3 local.db < backup.sql
